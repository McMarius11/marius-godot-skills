# Agent Feedback

Inbox für den self-improver. Der session-closer schreibt hier rein
wenn ein Agent etwas verpasst oder einen False Positive hatte.

Format:
```
- [YYYY-MM-DD] [Agent] VERPASST: [Was] | KONTEXT: [Details]
- [YYYY-MM-DD] [Agent] FALSE-POS: [Was] | KONTEXT: [Details]
```

Nach Verarbeitung durch self-improver wird der Eintrag mit ✅ markiert.
