# MCP Pro Issues

Probleme, fehlende Features und unerwartetes Verhalten von Godot MCP Pro.
Wird automatisch von Agents befüllt wenn MCP-Calls fehlschlagen oder unerwartet reagieren.

Nutze `/mcp-report` um eine saubere Zusammenfassung für den Entwickler zu generieren.

## Format
```
### [YYYY-MM-DD] [Kategorie] Kurztitel
- **Tool**: Tool-Name das betroffen ist
- **Erwartet**: Was hätte passieren sollen
- **Tatsächlich**: Was passiert ist (Fehlermeldung, falscher Rückgabewert, etc.)
- **Kontext**: In welcher Situation (welcher Agent, welches Projekt, welche Aktion)
- **Workaround**: Falls vorhanden
- **Status**: 🆕 Neu | 📧 Gemeldet | ✅ Gefixt
```

## Kategorien
- **FEHLT**: Ein Feature/Tool das es nicht gibt aber gebraucht wird
- **BROKEN**: Ein Tool das einen Fehler wirft oder nicht funktioniert
- **UNERWARTET**: Ein Tool das funktioniert aber nicht wie erwartet
- **VORSCHLAG**: Idee für ein neues Feature das den Workflow verbessern würde
